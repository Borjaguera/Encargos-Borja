Sobre mí

Hola, soy Borja Agüera, estudiante de Desarrollo de Aplicaciones Multiplataforma (DAM) y apasionado por la tecnología y el desarrollo de software.

Actualmente estoy aprendiendo y desarrollando proyectos tanto en mis estudios como por iniciativa personal. Me gusta crear herramientas útiles, páginas web y pequeñas aplicaciones que ayuden a resolver problemas reales.

En esta web puedes ver algunos de los proyectos que he realizado durante mi formación y otros que he desarrollado por interés personal para seguir mejorando mis habilidades.

También estoy abierto a colaborar en pequeños proyectos o desarrollar páginas web sencillas para personas, negocios o proyectos personales.

Intento ofrecer soluciones sencillas y accesibles, adaptándome a cada proyecto y a las necesidades de cada persona.

Tecnologías y lo que ofrezco

Actualmente estoy desarrollando mis habilidades como programador mientras estudio Desarrollo de Aplicaciones Multiplataforma (DAM).

Tecnologías que utilizo

  • HTML – estructura de páginas web
  • CSS – diseño visual y estilos
  • JavaScript – interacción en páginas web
  • Java – lógica de programación y aplicaciones
  • SQL – bases de datos

Qué puedo hacer

  • Crear páginas web informativas o personales
  • Desarrollar páginas para pequeños proyectos o negocios
  • Crear formularios de contacto o encargos
  • Conectar páginas web o aplicaciones con bases de datos
  • Diseñar páginas web claras y funcionales
  • Modificar o mejorar páginas web existentes

También tengo experiencia trabajando con bases de datos y vinculándolas a aplicaciones para gestionar información.
